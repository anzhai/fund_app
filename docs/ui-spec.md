## 1. 设计原则

### 1.1 核心原则

| 原则 | 说明 |
|------|------|
| 简洁专业 | 界面简洁，减少干扰，信息层次分明 |
| 数据优先 | 金融数据清晰展示，风险提示醒目 |
| 易于操作 | 核心交易流程不超过3步完成 |
| 一致性 | 同一产品线保持视觉与交互一致 |

### 1.2 设计价值观

- **可信赖**: 专业金融感，建立用户信任
- **高效**: 快速完成任务，减少输入成本
- **安全**: 重要操作需确认，风险提示清晰

## 8. 适配规范

### 8.1 屏幕适配

| 屏幕宽度 | 布局策略 |
|---------|---------|
| < 375px | 紧凑布局, 字号-2px |
| 375-414px | 标准布局 |
| > 414px | 宽松布局, 可放大元素 |

### 8.2 安全区域

iPhone X及以上设备需适配底部安全区:

```
┌─────────────────┐
│     StatusBar    │ 44px
├─────────────────┤
│                  │
│     Content      │
│                  │
├─────────────────┤
│   TabBar + Safe  │ 49px + 34px
└─────────────────┘
```

## 10. 页面流程图

### 10.1 登录流程

```
启动App → 检查登录状态
              ↓
    ┌─────────┴─────────┐
    ↓                   ↓
未登录 ──────────→ 已登录
    ↓                   ↓
显示登录页 ────→ 显示首页
```

### 10.2 首次交易流程

```
登录 → 检查开户状态
              ↓
    ┌─────────┴─────────┐
    ↓                   ↓
未开户 ──────────→ 已开户
    ↓                   ↓
引导开户 ───┬──→ 检查风评
            │           ↓
            │   ┌─────┴─────┐
            │   ↓           ↓
            │ 未做风评 ──→ 已风评
            │   ↓           ↓
            │  引导风评     交易
            │       ↘       ↓
            │        引导确认
            │           ↓
            └───────→ 交易
```

### 10.3 基金购买流程

```
选择基金 → 基金详情 → 买入
                            ↓
                      输入金额
                            ↓
                ┌───────────┴───────────┐
                ↓                       ↓
            风评未做                  风评正常
                ↓                       ↓
            引导风评              适当性校验
                │                       ↓
                │           ┌─────────┴─────────┐
                │           ↓                   ↓
                │       匹配                  不匹配
                │           ↓                   ↓
                │         确认              查看详情
                │           ↓                   ↓
                └─────── 确认  ──────────→ 继续交易
                                        │
                                        ↓
                                  选择支付方式
                                        ↓
                                  确认交易
                                        ↓
                                    交易成功
```
