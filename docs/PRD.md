## 1. 产品概述

### 1.1 项目背景

基金理财应用是一款面向个人投资者的移动端基金交易与管理平台，支持基金认购、申购、赎回、定投等核心交易功能，同时提供账户管理、风险评估、资产配置等理财服务。

### 1.2 目标用户

| 用户类型 | 特征描述 |
|---------|---------|
| 直销用户 | 直接在平台注册的用户，支持15位/18位身份证 |
| 代销用户 | 通过代销渠道注册的用户，未在APP注册手机号 |
| 纯代销用户 | 未做风险评估，以安全型客户显示 |
| 专业投资者 | 默认进取型，不能做风险评估 |

### 1.3 核心价值

- 便捷的基金交易体验
- 全面的风险评估与适当性管理
- 灵活的资产配置与组合管理
- 丰富的积分会员体系

## 3. 用户类型与权限

### 3.1 用户类型定义

| 用户类型 | 风险等级 | 可交易产品 | 限制说明 |
|---------|---------|-----------|---------|
| 直销用户(已风评) | 安全型/保守型/稳健型/积极型/进取型 | 匹配风险等级基金 | 正常交易 |
| 直销用户(未风评) | 安全型 | 仅安全型基金 | 需做风评 |
| 直销用户(风评过期) | 安全型 | 仅安全型基金 | 需重做风评 |
| 代销用户 | 安全型 | 仅安全型基金 | 需做风评 |
| 专业投资者 | 进取型 | 所有基金 | 不能做风评 |
| 黑名单用户 | - | 禁止交易 | 反洗钱/直销黑名单 |

### 3.2 证件有效期校验

| 年龄段 | 证件有效期 |
|--------|---------|
| 16周岁（不含）以下 | 5年 |
| 16周岁（含）-26周岁（不含） | 10年 |
| 26周岁（含）-46周岁（不含） | 20年 |
| 46周岁（含）以上 | 长期 |

## 4. 交易限制规则

### 4.1 适当性匹配

| 限制类型 | 处理方式 |
|---------|---------|
| 投资期限不匹配 | 修改投资期限 |
| 风评未做 | 引导做风评 |
| 风评已过期 | 引导重新测评 |
| 证件已过期 | 引导更新证件 |
| 客户信息不完善 | 引导完善信息 |
| 风险不匹配 | 查看详情/继续交易 |

### 4.2 大额交易

| 金额 | 要求 |
|------|------|
| 单日累计≥20万 | 双因素验证（短信+密码） |

### 4.3 支付方式

| 支付方式 | 说明 |
|---------|------|
| 瑞钱包支付 | 不能超过钱包余额 |
| 银行卡支付 | 受单笔限额限制 |
| 一步汇 | 转账到钱包 |

## 7. 业务流程图

### 7.1 用户注册登录流程

```
未注册用户 → 快速注册 → 设置密码 → 完成
                ↓
           已注册用户 → 登录方式选择
                ↓
    ┌─────────┼─────────┐
    ↓         ↓         ↓
手机号+密码  身份证+密码  短信验证码
    ↓         ↓         ↓
    └─────────┼─────────┘
              ↓
         登录成功
```

### 7.2 基金购买流程

```
用户登录 → 风评校验 → 产品展示 → 选择基金
    ↓                              ↓
风评未做 ──────────────────→ 风险匹配校验
    ↓                              ↓
引导做风评                   风险匹配? ──→ 不匹配 → 确认继续
    ↓                              ↓
继续交易 ──────────────────→ 选择支付方式
    ↓                              ↓
瑞钱包/银行卡 ────────────→ 确认交易
    ↓                              ↓
交易成功 ────────────────→ 生成订单
```

### 7.3 开户流程

```
用户注册 → 上传证件照 → OCR识别 → 信息确认
    ↓                              ↓
填写信息 ← ← ← ← ← ← ← ← ← ← ← ← ←
    ↓
银行卡绑定 → 设置交易密码 → 实名认证 → 开户成功
```
